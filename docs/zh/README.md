## 启动项目

```bash
# 克隆项目
git clone https://github.com/Marco-hui/vue-admin-web.git

# 安装依赖
npm install

# 启动开发环境
npm run dev
```

::: warning 注意
请确保你的 Node.js 版本 >= 8.6。
:::
